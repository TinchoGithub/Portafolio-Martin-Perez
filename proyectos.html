Rick and Morty WIKI

React Js | React-bootstrap

Aplicación Web (SPA) desarrollada con React JS que muestra los personajes de la serie Rick an Morty. En la misma utilizo los Hooks de React, utilizo axios para los llamados a la API REST de Rick and Morty, hago uso de react-router-dom para manipular la navegacion entre el paginado (Home, Registro, Login). Ademas utilizo react-hook-form para la construccion del formulario y Firebase como base de datos para guardar la informacion de los usiarios y la autenticacion para logearse. Tambien hago uso de react-bootstrap para darle estilos.
